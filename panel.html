دوره هایی که ثبت نام کرده اید

سیر تا پیاز Html & Html5 بخش اول متخصص طراحی سایت : زبان نشانه گذاری HTML به عنوان اولین پایه برای هر دوره آموزش طراحی سایت در نظر گرفته می‌شود. این زبان چیست و چه کاربردی دارد؟ HTML مخفف عبارت Hypertext Markup Language است.
اموزش پروژه محور Css و Css3 بخش دوم متخصص طراحی سایت : CSS یکی از زبان های پرکاربرد در طراحی صفحات وب می باشد که در حقیقت وظیفه استایل دهی به صفحات را دارد. CSS کوتاه شده عبارت Cascading Style Sheets بوده و در کنار HTML برای طراحی صفحات وب لازم است.
آموزش Bootstrap 4 بخش اول متخصص طراحی سایت : بوت استرپ (bootstrap) یک فریم ورک است که در طراحی صفحات وب به صورت رسپانسیو به کار می ورد. بوت استرپ شامل دستورات HTML، CSS و توابع جاوااسکریپت برای تولید و نمایش فرم ها، دکمه ها و المان های مورد نیاز طراحی وب می باشد.
آموزش javascript – jquery – ajax بخش چهارم متخصص طراحی سایت : جاوا اسکریپت برای اولین بار توسط شرکت Netscape و با نام LiveScript به عنوان نرم افزاری مفید جهت استفاده در دنیای وب به بازار عرضه شد ولی بعدها با حمایت شرکت Sun Microsystems (پدید آورنده JAVA ) با نام جاوا اسکریپت شناخته شد.
